# lateralus-compiler

Compiler construction toolkit in Lateralus — lexers, parsers, AST, IR, optimizations and code generation.

## Overview

This repository contains Lateralus (`.ltl`) source files. Lateralus is a modern, expressive programming language with a clean syntax inspired by Rust, Python and Swift.
